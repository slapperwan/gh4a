Смотрите GitHub и оставайтесь на связи со своей сетью участников. Следите за репозиториями и лучшими пользователями в GitHub. Просматривайте все действия пользователей, исходные коды и управляйте своими задачами и проблемами с помощью OctoDroid.

<b>Возможности</b>

Авторизация
* Авторизация веб-приложений OAuth2.
* Поддержка авторизации с помощью персонального токена доступа.

Общее
* Поддержка уведомлений.
* Поддержка нескольких профилей.
* Поддержка печати кодов, коммитов и прочего.
* Поиск по репозиторию, пользователю или коду.

Репозиторий GitHub
* Список репозиториев.
* Подписка на репозиторий.
* Просмотр веток и меток git (тегов).
* Просмотр запросов на извлечение.
* Просмотр соавторов/участников.
* Просмотр наблюдателей/сетей участников (networks).
* Просмотр задач и проблем.

Пользователь
* Просмотр базовой информации.
* Ленты активности.
* Подписаться/отписаться от пользователя.
* Просмотр публичных/отслеживаемых репозиториев.
* Просмотр подписчиков/подписок.
* Просмотр организаций (если тип — пользователь).
* Просмотр участников (если тип — организация).

Список задач и проблем
* Список задач и проблем (issues).
* Фильтрация по метке, исполнителю или этапу.
* Создание/редактирование/закрытие/повторное открытие задачи.
* Комментарий к задаче.
* Управление метками.
* Управление этапами (milestones).

Комментарий Git
* Просмотр утверждения git (коммита).
* Просмотр различий с подсветкой.
* Просмотр истории утверждений для каждого файла.

Обозреватель дерева/файлов
* Просмотр исходных кодов.
* Просмотр файлов с подсветкой синтаксиса кода.

Заметки Gist
* Список заметок gists.
* Просмотр gist.

Обзор GitHub
* Публичная хронологическая лента.
* Популярные репозитории (за сегодня, неделю, месяц).
* Блог Гитхаба.

Можно поделится:
* пользователем
* репозиторием
* утвеждением (коммитом)
* различием (diff)
* файлом
* задачей или проблемой (issue)
